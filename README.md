# sketchware-zh
sketchwarev7.0.0汉化版，sketchwarev7.0.0中文版

## 汉化版下载
https://github.com/eraycc/sketchware-zh/releases/download/v7-zh/Sketchware.Pro_v7.0.0.apk

## 使用string.xml
如果选择使用string.xml可下载官方sketchwarepro后，自行翻译官方原版[string.xml](https://github.com/Sketchware-Pro/Sketchware-Pro/blob/main/app/src/main/res/values/strings.xml)，或把本项目内的string.xml文件下载后移动到：内部存储/sketchware/localization/string.xml目录即可加载并使用，但是由于部分界面资源并没有链接到string.xml内，所以会存在很多未翻译部分（比如直接写在layout xml内的文本等），建议直接下载上方汉化版本，汉化更彻底
